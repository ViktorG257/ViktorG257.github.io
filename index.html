What is GIS?

Geographic Information Systems (GIS) is an umbrella term which incorporates all kinds of data sets, data management, and data processing required to create maps which represent spatial data, or geographic data in general. GIScience is the academic discipline which studies the applications, usage, an development of GIS.

GIS technology has many practical applications. By allowing for visual representation of spatial data, it allows the users to infer data correlations which might be difficult to visualise otherwise. Additionally, GIS data processing can allow for a unique combination and synthesis of arithmetic, scientific data, with the socio-economic data, which is seemingly impossible to do in a coherent manner at the first glance. Recently, it has been used in urban planning & development, sustainability, environmental sciences, geology, public healthcare, archaeology, precision agriculture, climate studies & meteorology, amongst other things. The urban planning involves the most standard application of mapping in simple instances, while precision agriculture utilises open satellite data to measure plant health using NDVI as their indicator, which allows the unhealthy plants to be replaced or cared for, and leaves the healthy plants alone, which reduces the usage of fertilizers and increases the crop yield.

Most common softwares used to apply GIS are ArcGIS, which is a private subscription-based software, and QGIS, which is a free open-source software. Both come with their own sets of advantages and disadvantages, as well as slightly different processing capabilities, so the software should be carefully chosen for each individual project.

Map Portfolio

Choropleth Maps of the World

Stationary and interactive world maps made with the help of QGIS and Leaflet. The maps display the UN sustainanable development goals (SDGs) by mapping their targets and indicator data for individual countries. The data was obtained from the official UN websites.

Analysis of Spatial Data in Utrecht

QGIS was used to obtain and present spatial data for Utrecht. An isochrone map was created to present the number of historical sites within a walking distance of a hospital. Additionally, VGI data was collected around the city, and processed using QGIS, although with limited success.

DEM of Western Northern Brabant

QGIS and ArcGIS Online were used to plot the digital elevation model (DEM) for the western part of the Northern Brabant region of the Netherlands. The result was compared to the map from 1730s to determine whether the river corridor had changed over time.

NDVI Maps of Croatia

QGIS and ArcGIS Online were used to process Sentinel 2 data from Google Earth Engine. A map was created to display the NDVI for the Medimurje region, as well as entire Croatia, in 2020 and 2025, along with the NDVI change in that time period.

Land Cover Classification via ML

ArcGIS Pro was used to determine the land cover classification (LCC) in Utrecht. The process was firstly done visually, and was then repeated using supervised and unsupervised machine learning to obtain different results.

Multicriteria Analysis

Analysis and discussion of multicriteria analysis and its applications for research purposes, as well as the applications for societal development. Exposition of the concept, as well its future potential. Includes a hypothetical workflow scheme.
